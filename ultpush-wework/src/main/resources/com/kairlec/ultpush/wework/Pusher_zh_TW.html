發送文本消息

# touser指定為消息要發送的人,必須為管理員才允許發送非自己,下文不再闡述
Post/Get %_SERVER_BASE_URL_%/text?userid={userid}&token={token}[&touser={touser}]

Request x-www-form-urlencoded:

key value 必須 備註
content {content} 是 消息內容，最長不超過2048個字節，超過將截斷

注:若為Get請求時,x-www-form-urlencoded里的key-value需放入QueryString里

發送文本卡片消息

Post %_SERVER_BASE_URL_%/textcard?userid={userid}&token={token}[&touser={touser}]

Request x-www-form-urlencoded:

key value 必須 備註
title {title} 是 標題，不超過128個字節，超過會自動截斷
description {description} 是 描述，不超過512個字節，超過會自動截斷
url {url} 是 點擊後跳轉的鏈接
btntxt {btntxt} 否 按鈕文字。 默認為「詳情」， 不超過4個文字，超過自動截斷

發送Markdown消息

Post %_SERVER_BASE_URL_%/markdown?userid={userid}&token={token}[&touser={touser}]

Request x-www-form-urlencoded:

key value 必須 備註
content {content} 是 markdown內容，最長不超過2048個字節，必須是utf8編碼

Markdown內容僅支持以下語法子集

標題 （支持1至6級標題，注意#與文字中間要有空格）

# 標題一
## 標題二
### 標題三
#### 標題四
##### 標題五
###### 標題六

加粗

**bold**

鏈接

[這是一個鏈接](http://work.weixin.qq.com/api/doc)

行內代碼段（暫不支持跨行）

`code`

引用

> 引用文字

字體顏色(只支持3種內置顏色)

<font color="info">綠色</font>
<font color="comment">灰色</font>
<font color="warning">橙紅色</font>

發送圖片

Post %_SERVER_BASE_URL_%/image?userid={userid}&token={token}[&touser={touser}]

Request multipart/form-data:

key value 必須 備註
file {file} 是 圖片文件,必須大於5B,不大於2M,支持JPG,PNG格式

發送語音

Post %_SERVER_BASE_URL_%/voice?userid={userid}&token={token}[&touser={touser}]

Request multipart/form-data:

key value 必須 備註
file {file} 是 語音文件,必須大於5B,不大於2M,播放長度不超過60s,僅支持AMR格式

發送視頻

Post %_SERVER_BASE_URL_%/video?userid={userid}&token={token}[&touser={touser}]

Request multipart/form-data:

key value 必須 備註
title {title} 否 視頻消息的標題,不超過128個字節,超過會自動截斷
description {description} 否 視頻消息的描述,不超過512個字節,超過會自動截斷
file {file} 是 視頻文件,必須大於5B,不大於10MB，支持MP4格式

發送普通文件

Post %_SERVER_BASE_URL_%/file?userid={userid}&token={token}[&touser={touser}]

Request multipart/form-data:

key value 必須 備註
file {file} 是 普通文件,必須大於5B,不大於20M
